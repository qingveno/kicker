Ming Commands Guide

This is a WIP so things might change
The bot prefix is -- (two dashes),
Required arguments are in [square brackets],
Optional arguments are in {curly brackets}.

Example: --command subcommand

Commands

friend friendreq

in-game only The bot sends you a friend request, it wont it's already in your friend list.

countdown cd

in-game only

discord

Generates the discord invite link.

mmstats {region}

Shows the number of people searching in the given region. See regions below

notes

Displays a pinned message left by an admin.

admins

Shows you the admin list.

stats

not implemented yet

restart

admin channel only

admin [command] [argument]

Admin specific commands, for subcommands and arguments see below

Regions

Usage of the mmstats command is --mmstats {region}
Generally you omit region and it will bring up default matchmaking regions.
However, if you wish to view all regions, write all as region: --mmstats all
